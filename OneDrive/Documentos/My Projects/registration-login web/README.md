# Login & Registration Form

A modern, responsive login and registration form built with HTML and CSS, featuring smooth animations and a beautiful gradient background.

## Features

- ✨ Modern glassmorphism design
- 🎨 Beautiful gradient background (pink sunset theme)
- 📱 Fully responsive layout
- 🔄 Smooth animations and transitions
- 🎯 Interactive form validation
- 🔒 Password and email input icons
- 💫 Hover effects and micro-interactions

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **Ionicons** - Beautiful icons for inputs

## Project Structure

```
login-registration-repo/
├── index.html          # Main HTML structure
├── style.css           # Styling and animations
└── README.md           # Project documentation
```

## Styling Features

### Background
- Linear gradient: `#f093fb` to `#f5576c` (sunset pink theme)
- Full viewport coverage

### Form Design
- Glassmorphism effect with backdrop blur
- Transparent background with border
- Smooth shadow effects

### Input Fields
- Floating label animation on focus
- Icon integration (email and password)
- Smooth color transitions
- Proper validation states

### Interactive Elements
- Hover effects on buttons
- Animated underline for navigation links
- Transform animations on interaction
- Focus states for accessibility

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/peterfficials6/login-registration-repo.git
   ```

2. Navigate to the project directory:
   ```bash
   cd login-registration-repo
   ```

3. Open `index.html` in your web browser

## Customization

### Changing Colors
- Modify the gradient in `style.css` (line 13):
   ```css
   background: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
   ```

### Adjusting Form Styling
- Form container: `.wrapper` class
- Input fields: `.input-box` class
- Buttons: `.btn` class
